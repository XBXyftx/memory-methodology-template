# Docs Lookup Workflow

## Purpose

A portable template for reading official docs through a local helper or direct documentation browser.

## Workflow

1. Search for the relevant topic.
2. Read the exact page.
3. Capture only the facts needed for the current task.
4. Reference the document path in your write-up.

## Notes

- Keep the command examples generic.
- Avoid storing environment-specific secrets or machine details.
